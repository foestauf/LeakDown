Realistic leakdown simulation for Derail Valley.

This mod simulates natural pressure loss in steam boilers (and, in future updates, brake systems). Steam engines in real life lose pressure over time due to small leaks and cooling—this mod brings that realism into the game!

✅ Features
Realistic steam boiler leakdown.

Configurable leak rate (0%–500% of realistic baseline).

Time-acceleration aware (scales to the game’s in-game day speed).

Modular design: will later support brake system leakdown.

🔧 Installation
Install Unity Mod Manager (UMM) (or your specific mod loader).

Download the latest LeakDown mod release.

Place the mod folder (or zip) into your game’s Mods directory.

Enable LeakDown in UMM.

⚙️ Configuration
Open the mod settings UI via Unity Mod Manager.

Use the slider to adjust the leakdown rate (expressed as a % of the real-world baseline of ~10% pressure loss per hour).

🚨 Requirements
Unity Mod Manager

Tested on Derail Valley version b99.4
